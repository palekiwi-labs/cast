# Goal
Implement an MCP tool for serving `cast` documentation. 

## Requirements
- Serve documentation files stored in the repository.
- Specifically support the new MCP command definition configuration (start with `docs/mcp/configuration.md`).
- Expose two tools: `list_documentation_entries` and `fetch_documentation_entry`.
- Target audience: AI agents using the MCP.
