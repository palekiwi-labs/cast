Manually test the client with Context7 server.

Example config for OpenCode

```
"mcp": {
  "context7": {
    "type": "remote",
    "url": "https://mcp.context7.com/mcp",
    "headers": {
      "CONTEXT7_API_KEY": "YOUR_API_KEY"
    },
    "enabled": true
  }
}
```

use `CONTEXT7_API_KEY` env variales as value
